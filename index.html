Laventa E-commerce website

The layout is fully responsive, ensuring smooth performance across all devices. Project 2

Student Management Counselling System

The Student Management Counselling System is a comprehensive web-based application developed using Java, JSF (JavaServer Faces), and MySQL to streamline student data management and counseling workflows. This system is designed to help educational institutions manage student profiles, track academic performance, and facilitate personalized counseling sessions. Using JSF, I created interactive and user-friendly web pages, while the backend logic in Java ensures robust functionality and data handling. MySQL was used to store and manage student records securely. This project helped me gain hands-on experience in building dynamic, database-driven applications and deepened my understanding of server-side development, user interface design, and real-world data management.

Project 3

Women Safety Web App

The Women Safety Web App is a responsive and user-focused application developed to enhance the personal safety of women using real-time features and alert mechanisms. Built with React.js for the frontend, this web app allows users to access emergency support through intuitive, time-based safety features. The interface is designed to be simple and fast, ensuring ease of use during critical situations. The app includes functionalities such as quick-alert buttons and location-based services (planned for future integration). This project highlights my skills in building responsive web applications with a focus on user experience, safety-oriented design, and readiness for backend integration to expand functionality in future versions.
